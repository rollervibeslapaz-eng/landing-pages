# Landing Template para GitHub Pages

Plantilla básica para vender landing pages a negocios locales.

## Cómo usar

1. Crea un repositorio nuevo en GitHub.
2. Sube estos archivos:
   - index.html
   - style.css
3. Entra a Settings > Pages.
4. En Branch selecciona `main`.
5. Guarda los cambios.
6. Espera unos minutos y GitHub te dará el enlace público.

## Qué cambiar para cada cliente

En `index.html` cambia:

- Nombre del negocio
- WhatsApp
- Servicios
- Paquetes
- Precios
- Textos
- Redes sociales

En `style.css` cambia:

- Colores
- Tamaños
- Estilos visuales
